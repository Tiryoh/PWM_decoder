# PWM_decoder

Sketch for PWM decoder for Arduino

[![Build Status](https://travis-ci.org/Tiryoh/PWM_decoder.svg)](https://travis-ci.org/Tiryoh/PWM_decoder)

## Requirements

* Arduino IDE > 1.0.6
* Arduino
  * Tested on Arduino UNO R3
